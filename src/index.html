Tethralink

I have built and maintained a plug and play developer tool I named ‘tethralink’. It serves as a wrapper over a standard array that uses multi-threaded solutions to iterate through the array faster. It includes helpful methods akin to what is commonly seen in more abstracted programming languages. It comes with a tradeoff of being more resource expensive while giving C++ developers breathing space so they can spend less time worrying about arrays
